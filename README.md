## Using the Interview Prep Agent

This repo includes an AI-powered interview prep system:

```bash
# Start a practice session
/interview-prep

# Run a timed mock interview
/mock-interview
```

**Features:**
- Senior/Staff FAANG mentor persona
- Asks you to explain your approach first
- Points out exactly where logic breaks
- Focuses on invariants and patterns, not memorization
